📄 Smart Resume Analyzer

Smart Resume Analyzer is a web application that evaluates resumes and highlights skill gaps for different job roles. The application simulates basic Applicant Tracking System (ATS) behavior by scanning resumes, detecting relevant skills, and suggesting improvements.

---

🚀 Live Demo
https://smart-resume-analyzer7.streamlit.app/


---

✨ Key Features

• Upload and analyze resumes in PDF format
• Detect relevant skills for selected job roles
• Identify missing skills and improvement areas
• Calculate resume strength score
• Compare resume content with job descriptions (ATS-style matching)
• Provide suggestions to improve resume quality
• Visual overview of skill coverage
• Downloadable resume analysis report

---

🛠️ Technologies Used

Python
Streamlit
Pandas

---

⚙️ How the Application Works

1. Upload a resume in PDF format
2. Select your branch and target role
3. (Optional) Paste a job description
4. The system scans the resume and generates insights such as:
   - Resume score
   - Detected skills
   - Missing skills
   - Suggestions for improvement
   - ATS match score
